# Red White and Blue
Shopify storefront for a progressive apparel brand.

[redwhiteandblue.com](https://redwhiteandblue.com)

---

## What It Does
- Custom Shopify theme built for a direct-to-consumer apparel store
- Product display, collection browsing, and cart optimized for conversion
- Fully responsive across mobile and desktop

## How It's Built
Custom Shopify theme built in Liquid with CSS and JavaScript. Component architecture follows Shopify's sections and blocks model, allowing modular layout control through the theme editor.

**Stack:** Shopify, Liquid, CSS, JavaScript
